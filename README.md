# Retail Sales Forecasting using Random Forest Regressor

## Overview

This project implements a Time Series Forecasting model using the CRISP-DM methodology to predict Weekly Sales based on historical retail sales data and several external factors.

## Dataset

The dataset consists of three files:

- sales data-set.csv
- Features data set.csv
- stores data-set.csv

The datasets are merged and prepared before being used for forecasting.

## Methodology

This project follows the CRISP-DM framework:

- Business Understanding
- Data Understanding
- Data Preparation
- Modeling
- Evaluation
- Deployment
- Conclusion

## Machine Learning Model

- Random Forest Regressor

## Evaluation Metrics

- RMSE
- MAE
- MAPE
- R² Score

### Result

- Accuracy : **96.35%**
- MAPE : **3.65%**
